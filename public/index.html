Features


Powerful dashboard

All of our servers come with a custom dashboard. It has a file manager, version installer and many other great features.

Free server page

Each server comes with a customizable website hosted by us. Add your own videos, pictures, text, or even blogs and forums and create a a hub for your community! Players can also vote for your server to keep it alive! (Minecraft server websites have votifier support)

Game library

We support various games ranging from Minecraft to CS:GO!
